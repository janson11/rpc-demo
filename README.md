1、核心原理

- 什么是RPC？

  RPC全称：Remote Procedure Call即远程过程调用。

- RPC作用：

  屏蔽远程调用跟本地调用的区别，让我们感觉就是调用项目的内的方法。

  隐藏底层网络通信的复杂性，让我们更专注于业务逻辑。
